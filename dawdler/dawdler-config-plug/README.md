# dawdler-config

## 模块介绍

统一配置中心

### 1. web端的pom中引入依赖

```xml
<groupId>dawdler</groupId>
<artifactId>dawdler-client-plug-config</artifactId>
```

### 2. dawdler服务端的pom中引入依赖

```xml
<groupId>dawdler</groupId>
<artifactId>dawdler-server-plug-config</artifactId>
```
